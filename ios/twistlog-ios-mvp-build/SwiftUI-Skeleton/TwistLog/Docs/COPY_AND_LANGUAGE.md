# TwistLog Copy And Language Rules

## Core Promise

```text
KNOW WHEN THE BOTTLE WAS OPENED.
```

## Safe Product Framing

Use:

- Bottle-opening tracker.
- Opening history.
- Opening recorded.
- Bottle opened.
- Last opening.
- Recent openings.
- Reminder nudges.
- Personal reference.

Avoid:

- Medication taken.
- Dose confirmed.
- Adherence confirmed.
- Prevents overdose.
- Missed dose.
- Medical advice.
- Treatment recommendation.

## App Microcopy

### Success

```text
Opening recorded.
```

### Manual Event

```text
Bottle opened
Manual
```

### NFC Future Event

```text
Opening recorded by NFC.
```

### Sensor Future Event

```text
Opening recorded by sensor.
```

### Reminder

```text
Time to check your bottle.
Open TwistLog to view recent openings.
```

Reminder settings copy:

```text
Reminders are local to this iPhone. TwistLog reminds you to check a bottle; it does not confirm medicine was taken.
```

Test reminder body:

```text
Test reminder: Open TwistLog to view recent openings.
```

### Safety

```text
TwistLog records bottle-opening events for personal reference and reminders. It does not verify that medicine was taken and is not medical advice.
```

### Recent Opening Warning

```text
Recent opening found.
This bottle was already opened at 8:14 AM.
Record anyway
Cancel
```

## Color Semantics

- Green = TwistLog identity, healthy/normal app state.
- Orange = opening was recorded.
- Red = warning or problem.
- Gray = inactive, no recent event, unknown.

Do not use orange for warnings.
